# 🚀 Crypto Dashboard

A modern crypto dashboard UI that shows live prices of Bitcoin, Ethereum, and Solana.

## 🔥 Features
- 📊 Live crypto prices (CoinGecko API)
- 📈 Chart visualization
- 💻 Clean dashboard UI
- 🔄 Auto-refresh every 10 seconds

## 🖥️ Demo
https://your-username.github.io/crypto-dashboard/

## ⚙️ Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## 📦 Installation

1. Clone the repo:
git clone https://github.com/your-username/crypto-dashboard.git

2. Open folder:
cd crypto-dashboard

3. Run:
Just open index.html in browser

## 🌐 API Used
CoinGecko API (Free)
